# REH DOGG GP – 3D Arcade Racing

**REH DOGG GP** is a fast-paced, browser-based 3D racing game featuring 5 unique global stages and lap-based progression. Built with Three.js, it delivers arcade-style driving with simple keyboard controls, dynamic environments, and a retro neon aesthetic.

## 🎮 Features

- **5 distinct stages** – Tokyo Neon, Sahara Desert, Alpine Snow, Forest Ridge, Volcano Circuit  
- **3 laps per stage** – complete all laps to advance to the next track  
- **Full 3D graphics** – real-time lighting, shadows, and a detailed car model  
- **Arcade physics** – drift, accelerate, brake, and steer with responsive controls  
- **Dynamic UI** – stage name, lap counter, and best lap time display  
- **Restart button** – reset the current stage at any time  
- **Keyboard controls** – Arrow keys or WASD  
- **No installation** – runs directly in any modern web browser

## 🚀 How to Play

1. **Download or clone** this repository.  
2. Open `index.html` in a modern browser (Chrome, Edge, Firefox, Safari).  
3. Use the keyboard to control your car:  
   - **↑ / W** – Accelerate  
   - **↓ / S** – Brake / Reverse  
   - **← / A** – Steer left  
   - **→ / D** – Steer right  

4. Complete **3 full laps** on the current stage to unlock the next stage.  
5. After finishing all 5 stages, you become champion and the game restarts from stage 1.  
6. Click the **RESTART** button at any time to reset the current stage (laps go back to 0, same stage).

## 🛠️ Built With

- [Three.js](https://threejs.org/) – 3D graphics engine  
- HTML5, CSS3, JavaScript (ES6)  
- No external build tools – pure vanilla implementation
